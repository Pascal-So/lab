This is the new BIT-101 LAB.

I plan to post regular visual code experiments here. Shooting for daily, but at least several per week.

You'll be able to find the code here: https://github.com/bit101/lab

A lot of the code will use some of my personal JavaScript libraries.

The bitlib library is a collection of useful functions for manipulating HTML Canvas, math functions, a random number generator, some animation and color stuff. This will grow and change over time.

The QuickSettings library creates a panel with various types of controls for changing values in real time.

Feel free to use any of these in your own projects.
